# Plezel::Ruby

Ruby binder for Plezel API.

## Installation

Add this line to your application's Gemfile:

    gem 'plezel-ruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install plezel-ruby

## Usage

Set your Plezel API key using

    Plezel.api_key = YOUR_API_KEY
